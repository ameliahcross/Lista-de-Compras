OBJETIVO:

Crear un programa que almacene listas de compras, dividir las compras por categoría ejemplo; superMercado, ropas y calzados

    - Se debe agregar, eliminar, editar y listar en cada sección. 
    - Debe de tener un login con una lista de usuarios para entrar
    - Si al realizar el login no existen los nombres preestablecidos en la lista de usuarios, debe notificarle al usuario
    - Debe validar que no se introduzcan valores vacíos a las listas ni al usuario

    Nota: la aplicación debe tener un menú principal después de ingresar, para navegar entre las categorías 

    La aplicación debe ser en consola.
